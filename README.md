# 农业无人机智能作业调度平台

前端使用 Vue3 + Vite，后端使用 Spring Boot 3 + SpringMVC + MyBatis-Plus，数据库使用 MySQL。系统覆盖登录鉴权、作业驾驶舱、地图态势、无人机管理、地块管理、任务管理、用户管理、调度计划、操作日志、AI 虫害检测十大功能模块。

## 技术亮点

- 多因素调度评分：综合任务优先级、虫害风险、截止时间、电量、载荷和喷幅生成调度方案，输出 0-100 评分和可解释决策理由。
- AI 虫害检测：基于 HSV 颜色空间分析农田照片，识别健康、黄化、坏死、真菌、锈病五类区域占比，输出病害名称、风险等级和置信度。
- 数据看板：作业驾驶舱聚合 8 个信息面板，包括指标卡片、无人机资源池、任务状态、状态分布、电池健康、执行进度、虫害概览和操作动态，GSAP 动画增强交互体验。
- 前后端分离：Vue3 工作台通过 REST API 调用 Spring Boot 服务。
- SpringMVC：使用 Controller、RESTful 映射、全局异常处理、跨域配置和接口访问拦截器完成 Web 层处理。
- MyBatis-Plus：使用 BaseMapper 完成核心 CRUD，减少模板代码。
- 可解释决策：每条调度计划保留评分和分配原因，便于答辩和简历描述。
- 地图态势：维护地块边界点、无人机经纬度和作业轨迹点，前端展示地块多边形、无人机位置和轨迹线，支持高德地图 JSAPI 2.0 和内置 SVG 两种渲染模式。
- 操作留痕：记录登录、调度方案生成、确认、取消、开始作业、完成作业等关键动作。

## 本地运行

1. 创建数据库并导入样例数据：

```sql
source database/schema.sql;
```

已有数据库只想升级新增登录和作业执行表时，执行：

```sql
source database/upgrade_auth_operation.sql;
```

默认登录账号：

```text
admin / admin123
```

2. 修改后端数据库账号：

```yaml
backend/src/main/resources/application.yml
spring.datasource.username: root
spring.datasource.password: 你的数据库密码
```

3. 启动后端：

```bash
cd backend
mvn spring-boot:run
```

4. 启动前端：

```bash
cd frontend
npm install
npm run dev
```

如需启用高德地图，在 `frontend` 目录创建 `.env`：

```text
VITE_AMAP_KEY=你的高德Web端JSAPI Key
VITE_AMAP_SECURITY_CODE=你的高德安全密钥
```

重启前端后进入“地图态势”，可查看地块边界、无人机位置、作业轨迹，并点击地图读取经纬度。

5. 浏览器访问：

```text
http://localhost:5173
```

## 功能模块

### 1. 登录鉴权
基于 JWT 的 Token 认证机制，登录后前端将所有请求携带 `Authorization: Bearer <token>` 头。后端通过 `AuthInterceptor` 拦截 `/api/**` 路径进行身份校验，排除登录接口和静态资源。支持多角色（管理员/操作员），不同角色在任务管理、用户管理等模块中拥有不同数据可见范围。

### 2. 作业驾驶舱（数据看板）
系统首页，聚合展示作业全局态势：
- **指标卡片**：无人机总数、空闲数量、待调度任务数、今日计划作业亩数，数字带 GSAP 滚动动画
- **无人机资源池**：全部无人机列表，含型号、电量进度条、工作状态标签
- **任务状态统计**：待调度 / 已调度 / 已生成计划的水平柱状图
- **无人机状态分布**：空闲 / 作业中 / 充电中 / 维护中四类占比
- **电池健康**：平均电量展示 + 低电量无人机（<25%）预警列表
- **今日执行进度**：计划面积 vs 实际完成面积对比，含完成率进度条
- **虫害检测概览**：高/中/低风险检测数量卡片 + 最近检测记录列表
- **最近操作动态**：时间线形式展示最新 8 条系统操作日志

### 3. 地图作业态势
支持两种地图模式：
- **高德地图模式**：配置 API Key 后，使用 AMap JSAPI 2.0 加载真实地图，展示地块多边形、无人机 Marker、作业轨迹折线，支持轨迹回放动画
- **内置 SVG 地图模式**：未配置 Key 时自动降级，基于经纬度相对位置渲染地块、无人机、轨迹

地图右侧经纬度面板可点击读取坐标，轨迹回放时实时更新位置和电量信息。

### 4. 无人机管理
管理所有无人机设备，支持新增、编辑、删除。核心字段包括编号、型号、载荷（kg）、喷幅（m）、电量百分比、工作状态（空闲/作业中/充电中/维护中）、经纬度位置。列表展示全部无人机并可筛选。

### 5. 地块管理
管理农田地块信息，支持新增、编辑、删除。字段包括地块名称、作物类型、面积（亩）、土壤墒情（低/中/高）、虫害风险等级（低/中/高）、经纬度坐标。虫害风险等级可与 AI 虫害检测结果联动更新。

### 6. 任务管理
管理作业任务，支持新增、编辑、删除。字段包括任务编号、关联地块、作业类型（植保喷洒/变量施肥/播撒补种/巡田监测）、作业面积、优先级（1-5）、最早开始时间和最晚截止时间、状态（待调度/已调度）。支持按状态筛选和分页浏览。

### 7. 用户管理
管理员可查看、新增、编辑、删除系统用户，操作员仅可查看和修改自身信息。字段包括用户名、真实姓名、角色（管理员/操作员）、联系方式。支持密码修改功能。使用 Vue Router 独立路由实现。

### 8. 调度计划
系统核心模块，"生成调度方案"按钮触发多因素评分算法：
- 综合**任务优先级**、**虫害风险等级**、**截止时间紧急度**、**无人机电量**、**载荷与喷幅能力**、**无人机到地块距离**和**是否超时**共 7 个维度
- 为每个待调度任务计算所有可用无人机的匹配分数（0-100），选择最高分无人机分配
- 每条计划保留评分（score）和决策理由（decisionReason），实现可解释调度

调度计划全生命周期管理：生成 → 确认 → 开始作业 → 完成作业 / 取消。下方展示作业执行记录表（任务号、地块、无人机、开始/完成时间、实际面积、耗时、结果）。

### 9. 操作日志
记录系统中所有关键操作，包括登录、调度方案生成、确认、取消、开始作业、完成作业等。列表展示最近 100 条，字段含时间、模块名称、动作名称、操作对象和日志内容，支持追溯审计。

### 10. AI 虫害检测
基于 HSV 颜色空间分析的农作物病虫害检测：
- 上传农田照片（JPG/PNG，最大 10MB）
- 后端 `ColorBasedDetectionEngine` 分析绿色（健康）、黄色（黄化）、棕色（坏死）、灰色（真菌/白粉）、橙色（锈病）五种颜色占比
- 输出检测病害名称、风险等级（高/中/低）、置信度和分析描述
- 支持将检测结果的风险等级应用到关联地块，自动更新地块虫害风险
- 检测历史记录支持查看详情、应用到地块和删除操作

## 系统简要总结

农业无人机智能作业调度平台：基于 Spring Boot、SpringMVC、MyBatis-Plus、MySQL、Vue3 实现农田作业任务管理、无人机资源管理和调度方案生成。设计多因素调度评分策略，结合任务优先级、作业窗口、虫害风险、电量、距离、载荷与喷幅进行资源匹配，并输出可解释调度原因；前端实现作业驾驶舱、地图态势、AI 虫害检测和调度结果可视化，支持高德地图集成与轨迹回放。
